# AutoPhyloR
Pipeline filogénico automatizado con acceso a NCBI

Para utilizar la herramienta descargue los paquetes, elimine el archivo "README" o léeme en español e importelos a su archivo en blanco en R
Desde su archivo en blanco, ejecute la aplicación entrando a "app.R" y dando click arriba en "Run app"
Corra la aplicación y escribanos en caso de cualquier problema.
